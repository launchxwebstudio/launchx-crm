How to Onboard a New Client
1

Client signs contract + pays $75 deposit

Send the contract PDF via email. Once deposit is received via Stripe, proceed immediately to step 2.

2

Buy a Twilio number for the client

Go to Twilio Console → Buy a Number. Search their local area code. Cost: $1.15/month. Set both webhook URLs above on the new number.

3

Add client in the Active Clients tab

Go to Active Clients and click Add Client. Enter their Twilio number, business name, booking link, and custom messages. Saves directly to Supabase — webhook picks it up instantly, no code needed.

4

Set up call forwarding

Client calls their carrier to forward unanswered calls to their new Twilio number. AT&T: dial *92 + number · Verizon: *71 + number. Their public number stays the same.

5

Test → go live → send Stripe recurring link

Call the forwarded number, don't answer, confirm the auto-text fires. Once live, send the Stripe recurring payment link. Free trial starts from this moment.
